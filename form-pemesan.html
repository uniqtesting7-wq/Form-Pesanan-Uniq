Setup Firebase Sync
Setup sekali saja — gratis selamanya
Setelah ini, semua perangkat yang buka URL yang sama akan sinkron otomatis secara real-time.
Langkah-langkah:
1. Buka console.firebase.google.com
2. Klik Add project → isi nama → klik Continue sampai selesai
3. Di dashboard project, klik ikon </> (Web app)
4. Isi nama app → klik Register app
5. Copy nilai firebaseConfig yang muncul
6. Paste di kolom bawah ini
7. Di sidebar kiri: Build → Realtime Database → Create database
   Pilih Start in test mode → Enable
Config disimpan di browser, tidak perlu isi ulang
Kirim ke Google Sheets
Setup sekali saja:
1. Buka Google Sheets → buat spreadsheet baru
2. Klik Extensions → Apps Script
3. Hapus kode default, paste script di bawah
4. Klik Deploy → New deployment → Web app
5. Execute as: Me  |  Who has access: Anyone
6. Klik Deploy → copy URL → paste di kolom bawah
URL disimpan otomatis, tidak perlu isi ulang
